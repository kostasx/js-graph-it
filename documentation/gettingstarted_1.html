Getting started

Setup

<html>
  <head>
    <script type="text/javascript" src="js-graph-it.js"></script>
    <link rel="stylesheet" type="text/css" href="js-graph-it.css">
  </head>
  <body>
  <script>
  window.onload = function(){ initPageObjects(); }
  </script>
  </body>
</html>

The first thing to do to use js-graph-it in your html page is to import the js-graph-it.js file.

The other file js-graph-it needs to work properly is js-graph-it.css, this must be declared this way:

In order to activate js-graph-it, the only javascript code required is the invocation of the function initPageObjects(). Place it into the onload attribute of the window object, inside your script.
